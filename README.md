## 运行方法

首先在根目录下新建文件 `.env`，随后根据 `.env.template` 的模版指定各环境变量。

- 安装依赖：执行 `./scripts/prebuild.sh`
- 编译构建：执行 `./scripts/build.sh`
- 启动：执行 `./scripts/start.sh`
- 停止：执行 `./scripts/stop.sh`
